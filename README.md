# Logic Arena- Number Guessing Game.

#Description:
Logic Arena is a simple number guessing game built using HTML,CSS and JAVASCRIPT.
The Player tries to guess a randomly generated number between 1 and 50 . The game provides feedback if the guess is too high or too low, counts the number of attempts, and allows player
to reset the game at any time.


#Features#
Input field to enter a guess.
Submit button to check the guess.
Feedback for "Too high, or Too low".
Attempts counter.
Reset button to start a new game.
Clean and user friendly UI.
Error handling for invalid Inputs.


#How to Play
Enter a number between 1 and 50 in the input field.
Click the guess button.
Read the Feedback:
"Too high"= Your guess is higher than the target number.
"Too low" =your guess is lower than the target number.
"You Won" = Correct guess.
The number of attempts is displayed below the feedback.
click reset  to start a new game.

#Technologies Used
HTML
CSS
JAVASCRIPT


#Game Logic
Generate a random Number between 1 and 50.
Store the number of attempts.
Validate the input
Compare the guess with the target number.
Display feedback


#On clicking reset
-Generate a new random nuumbers 
-Reset attempts to 0
-Clear input and feedback.
